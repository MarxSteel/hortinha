URL access to JSON data

  • EatLocal/api/<type> will return the json for all rows in the dataset matching <type>

Valid <type> Values

  • All - all rows in the dataset, this is the same as the content of the json file download
  • Agritourism - only rows in the dataset that represent agritourism-related vendors
  • CSA - only rows in the dataset that represent community supported agriculture vendors
  • Farmers Market - only rows in the dataset that represent farmers' markets
  • Restaurant - only rows in the dataset that represent restaurants
  • Winery - only rows in the dataset that represent wineries
